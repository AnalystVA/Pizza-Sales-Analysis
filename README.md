# Pizza-Sales-Analysis
This project analyzes pizza sales data using SQL and Power BI to identify sales trends, product performance, and key business KPIs.

## Overview
This project transforms raw pizza sales data into a clean and interactive dashboard. It helps track revenue, orders, product performance, and customer buying patterns in one place
## Objective
The goal of this project was to create a dashboard that supports data-driven decision-making by showing what sells best, when sales peak, and which products need attention.

## Tools Used
- Power BI
- SQL
- Excel
## Key Metrics
- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Average Pizzas per Order

## Dashboard Features
- KPI cards for high-level performance tracking
- Sales trends by day, month, or other time periods
- Category-wise sales analysis
- Size-wise sales analysis
- Best-selling and worst-selling products
- Interactive filters and slicers
## Business Insights
- Identifies top-performing pizzas and product categories
- Reveals peak sales patterns
- Helps understand customer demand and ordering behavior
- Supports faster and better business decisions
## Project Outcome
This dashboard turns raw sales data into actionable insights that are easy to explore and interpret.
